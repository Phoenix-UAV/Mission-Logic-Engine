Custom Mission Planning Interface.

## Instructions:

- Make a branch with title your name and the part of the project you are working on.
- work only on your branch main will be protected.

## Setting up MAVProxy
Currently, only a Bash script is available for simplifying MAVProxy setup.

### Using the Bash Script
To use the script:
1. Change the variable `uav` to the directory connected to the UAV telemetry.
2. Go into the directory `MAVProxy`.
3. Run `./mavinit.sh`.
